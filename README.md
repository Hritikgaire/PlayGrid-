PlayGrid is a C-based mini game suite featuring classic games with a fun twist. It demonstrates procedural programming.

 #Included Games
1.Tic-Tac-Toe – Play against a basic AI bot
2.Random Maze – Navigate through a randomly generated maze
3.Hangman – Guess the hidden word before it’s too late
4.Minecone – A Minesweeper-inspired game
 #Features
1.Written entirely in C using procedural programming
2.Minimal AI usage (≈1%) for Tic-Tac-Toe
3.Simple, easy-to-understand code, perfect for learning and experimentation
